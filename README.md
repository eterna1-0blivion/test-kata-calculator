# Java Calculator

* Performs the simplest operations with Arabic `(1, 4, 9)` and Roman `(I, IV, IX)` numerals.
* Accepts simple operators `(+, -, *, /)` and numbers from `1` to `10`.
* Fractional remainder is discarded.
* Output value of Roman numerals cannot be less than `I (1)`.
